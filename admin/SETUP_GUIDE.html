2. 本番サーバーへのアップロード

  1. FTPソフト（FileZilla等）またはcPanelのファイルマネージャーを使用
  2. 全ファイル・フォルダをサーバーのドメインルート（public_html/等）にアップロード
  3. ブラウザで https://yourdomain.com/ を開いて表示を確認
⚠ 注意： admin/ フォルダはパスワード保護をかけることを推奨します。
サーバーの .htaccess で Basic認証を設定するか、サーバー管理画面のディレクトリ保護機能をご利用ください。

3. 管理者ログインと編集の使い方

ログイン方法

  1. サイトのフッター右下の Admin リンクをクリック
  2. 管理者IDとパスワードを入力してログイン
  3. ダッシュボードで各ページへのリンクを確認

デフォルト認証情報

ID: admin　　Password: biei2026!
※ 初回ログイン後、ダッシュボードの「パスワード変更」から必ず変更してください。

テキスト編集方法

  1. 編集したいページを開く（ログイン済みの場合、画面下部に管理バーが表示）
  2. EDIT PAGE ボタンをクリック → 編集可能な要素がハイライト表示
  3. テキストをクリックして直接編集
  4. SAVE CHANGES をクリックして保存

画像の差し替え方法

  1. 編集モードをONにする
  2. 差し替えたい画像エリアをクリック → ファイル選択ダイアログが開く
  3. 新しい画像ファイルを選択 → 自動的に保存される
💡 推奨画像サイズ:
ヒーロー画像: 1920×1080px 以上（横長）
各セクション画像: 1200×800px 程度（4:3 または 16:9）
形式: JPG（写真）、PNG（ロゴ・透過）、WebP（高圧縮）

4. 画像・動画の追加方法（本番環境）

  1. 画像ファイルを images/ フォルダにFTPでアップロード
  2. HTMLファイルの <img src="images/your-photo.jpg"> に変更
  3. 動画の場合は <video src="videos/hero.mp4" autoplay muted loop playsinline></video> で埋め込み

5. お問い合わせフォームの本番設定

現在の reserve.html のフォームは、JavaScriptでのバリデーションのみです。
実際にメールを送信するには、以下のいずれかで対応してください：

  • 推奨: Formspree — https://formspree.io に登録し、action="https://formspree.io/f/XXXX" を設定
  • レンタルサーバーのPHPメール送信（mail()関数）
  • サードパーティフォームサービス（Googleフォーム埋め込み等）

6. データの保存について

現在のCMSはブラウザの localStorage を使用しています。

  • ✅ 同じブラウザでアクセスした場合：保存したデータが反映される
  • ✅ 別のブラウザやデバイスでも見える：HTMLファイル自体を書き換えた内容
  • ⚠ 注意：CMSでの編集はそのブラウザのlocal storageに保存されます
本格運用時の推奨：
WordPressやHeadless CMS（microCMS、Contentful等）への移行、またはPHP+MySQLバックエンドの追加をお勧めします。 ご相談ください。
